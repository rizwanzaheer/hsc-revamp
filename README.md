# VueJS 3 + Vite + Vuex + Tailwind CSS 


## Stack

-   [Vue 3.0](https://v3.vuejs.org/)
-   [Vuex 4.0](https://vuex.vuejs.org)
-   [Router](https://router.vuejs.org/)
-   [Tailwind CSS](https://router.vuejs.org/)
-   eslint
-   prettier

## Install

```
git clone github/bitbucket url
cd projectname
npm install
```

## Run / Develop

```

npm run dev

```

## Build

```

npm run build

```

## Serve / host

```

npm run serve

```
